FUNGSI DARI GIT ADD, GIT COMMIT, DAN GIT PUSH

1. git add = untuk memindahkan perubahan pada berkas dari direktori kerja
ke staging area
2. git commit = menyimpan snapshot perubahan yang telah dibuat ke dalam riwayat 
repositori git
3. git push = untuk mengunggah commit dari repositori lokal ke repositori jarak jauh
